Deep Knowledge Tracing

Deep Knowledge Tracing (DKT) uses the Long Short Term Memory (LSTM) model to the problem of predicting student responses to exercises based on their past activity. This models the knowledge state of a student and predicts whether a student has sufficient knowledge to answer problems not yet seen. The learned model can be used for intelligent curriculum design based on the current knowledge state of a student, resources can be suggested to students based on their individual needs, and content that is predicted to be too easy or too hard can be skipped or delayed and can give necessary feedback.

AR-Dino

This is a marker based augmented reality dinosaur encyclopedia app, which augments dinosaurs onto the markers and explains each dinosaur with audio. This was built using Unity and Vuphoria SDK.

This can be used to display an animation of dinosaur to explain a concept given in a book right onto the book page itself. The page should have a marker.
